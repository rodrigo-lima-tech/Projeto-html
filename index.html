O que eu aprendi sobre html?

O curso de HTML da DIO.me ensina desde os conceitos básicos até os principais elementos para estruturar páginas web.

Estrutura de títulos e textos

ensina o uso das tags h1, h2, h3, h4, h5, h6, Essas Tags servem para definir títulos e subtítulos, com níveis de importância, sendo h1 o maior nível e os demais menores de forma decrescente como indica o exemplo:

h1 título

h2 subtítulo 1

h3 subtítulo 2

h4 subtítulo 3

h5 subtítulo 4
h6 subtítulo 5

também temos o uso das Tags Strong e itálico para dar ênfase em alguma parte ou palavra do texto.

Onde strong é ultilizada para dar ênfase forte e tem significado semântico de conteúdo importante.
Já a Tag itálico não tem tanto sentido semântico e é mais ultilizada pra citações e visual dos textos.

Tabelas:

Estruturas báscias

Em tabelas aprendemos a criar tabelas usando as tags table e tr para criar linhas td para criar células
e th para cabeçalhos

Organização e estilo

a parte semântica se encontra aqui também para organizar a criação de tabelas, geralmnente usando thead, tbody e tfoot
respectivamente para cabeçalhos, corpo e rodapés das tabelas.
a tag caption para o título da tabela e colgroup e col para formatar as colunas

Células

Links:

A utilização da tag a , também conhecida como âncora, para criar links que conectam documentos ou diferentes partes de uma mesma página. É o elemento fundamental de navegação na web.

O atributo href, fundamental para definir o destino do link.
um bom exemplo do seu uso está no índice.

Também temos o atributo target que define o link para outra url, seja para outra página web do mesmo site ou para outros sites
abrindo o link numa nova aba ou janela

Citações:

sobre as citações temos as tags q usadas para citações dentro de um parágrtafo
a tag blockquote usado para citações em blocos, geralmente em textos mais longos
e a tag cite usada para destacar a fonte, geralmente renderizada em ítalico

as tags q / cite são usadas dentro de blockquote
